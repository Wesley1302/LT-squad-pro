# AI Agent Product Checklist

- [ ] Resolve uma TAREFA específica (não conversa genérica)
- [ ] Usuário ideal e problema claros
- [ ] Inputs obrigatórios definidos
- [ ] Fluxo de execução + regras de decisão
- [ ] Formato de saída definido
- [ ] Limites + anti-alucinação
- [ ] Testes simulados de uso
- [ ] Funciona como PRODUTO low ticket (a persona pagaria por isso), não como brinquedo

**Decisão:** PASS → WLT-generate-ai-agent-prompt (skill assistant-creator-v2) · FAIL → recortar até 1 tarefa · BYPASS → frase exata
