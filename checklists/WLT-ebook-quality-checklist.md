# Ebook Quality Checklist

- [ ] Objetivo e prático (20-80 páginas), consumível em <48h
- [ ] Cada capítulo entrega 1 micro-resultado
- [ ] Sumário claro, sem enchimento teórico
- [ ] Exercícios / checklists presentes
- [ ] Resumo final acionável
- [ ] NÃO é livro inchado
- [ ] Linguagem na voz do avatar

**Decisão:** PASS → gerar prompt do ebook · FAIL → cortar teoria/capítulos · BYPASS → frase exata
