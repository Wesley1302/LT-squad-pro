# Demand Validation Checklist

- [ ] Procura confirmada (buscas / vídeos com views)
- [ ] Volume de comentários / engajamento no tema
- [ ] Produtos concorrentes existem
- [ ] Anúncios ativos no tema (ads library)
- [ ] Urgência / dor recorrente presente
- [ ] Potencial criativo (dá pra fazer criativo naturalmente)
- [ ] Potencial de produto instantâneo (<48h)
- [ ] 5 scores atribuídos (0-10) com fonte
- [ ] Decisão explícita: GO | NO_GO | PIVOT_ANGLE
- [ ] "Tem no YouTube grátis" NÃO foi usado para invalidar

**Decisão:** GO → segue · NO_GO/PIVOT → volta ao WLT-avatar-researcher (outro ângulo) · BYPASS → frase exata
