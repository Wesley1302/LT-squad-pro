# Funnel Diagnosis Checklist

- [ ] Métricas fornecidas (ROAS, CPV, CPC, CAC + secundárias)
- [ ] Janela válida (teste: 7d · convertendo: mês+semana+3d, não 1 dia)
- [ ] R3C lido de cima pra baixo (CPV → CPC → CAC)
- [ ] Primeiro número fora da meta ISOLADO como gargalo
- [ ] Culpado mapeado (CPV→criativo/público; CPC→página; CAC→oferta/checkout)
- [ ] Uma correção por vez
- [ ] Não mandou trocar produto sem teste válido
- [ ] Se ROAS<2 e quer escalar → bloqueou (corrigir rota, não escalar)

**Decisão:** PASS → rotear correção / WLT-scale-architect (se winner) · FAIL → pedir métricas · BYPASS → frase exata
