# Money Skill Checklist

- [ ] É skill (conhecimento APLICADO), não só conhecimento
- [ ] Tem evidência de resultado (já ajudou alguém / já foi pago ou elogiado)
- [ ] "Melhor que a média" nesse tema
- [ ] Não é hobby sem ninguém disposto a pagar
- [ ] Repetição/experiência real, não só diploma
- [ ] Sem evidência? Está marcada como HIPÓTESE, não como fato

**Decisão:** PASS → WLT-niche-compressor · FAIL → pedir prova de resultado externo · BYPASS → frase exata do usuário
