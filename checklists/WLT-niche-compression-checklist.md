# Niche Compression Checklist

- [ ] Micro-nicho tem os 3 elementos: [público específico] + [contexto] + [resultado]
- [ ] É específico o bastante para justificar preço maior
- [ ] Vantagem injusta nomeada (o que o usuário tem que os concorrentes não têm)
- [ ] Não é nicho amplo/genérico ("marketing", "emagrecimento")
- [ ] A dor do micro-nicho tem disposição a pagar

**Decisão:** PASS → WLT-avatar-researcher · FAIL → comprimir mais um nível · BYPASS → frase exata do usuário
