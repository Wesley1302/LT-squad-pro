# Initial Diagnosis Checklist

- [ ] user_stage identificado (from_zero | has_niche | has_product | has_traffic_no_sales | has_sales_wants_scale)
- [ ] Workflow selecionado coerente com o stage
- [ ] Nenhuma etapa avançada liberada sem pré-requisito
- [ ] Se o usuário pediu atalho, o bloqueio foi aplicado (risco -> pré-req -> redireciona)
- [ ] Resposta específica, não genérica

**Decisão:** PASS → roteia · FAIL → pergunta o dado ausente · BYPASS → só com "quero pular com risco documentado" (grava em journey.bypasses)
