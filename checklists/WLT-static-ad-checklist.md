# Static Ad Checklist

- [ ] Mínimo 5 estáticos/carrosséis na rodada (ideal 10+)
- [ ] Headline usa language_bank do avatar
- [ ] Ancora valor antes do clique (se mostra preço)
- [ ] Mistura de formatos (headline, carrossel, print, comparação, checklist, mockup, tabela, meme, advertorial)
- [ ] Prompt de imagem por criativo (1:1 / 4:5 / 9:16)
- [ ] Prova real (print com nome), nunca fabricada
- [ ] CTA leva pra página; nunca "Comprar"
- [ ] role_in_funnel preenchido; front-end não vende bump diretamente
- [ ] Alinhado com a oferta completa (principal + bumps + upsell)

**Decisão:** PASS → WLT-recording-director / gerar imagem · FAIL → gerar mais/ajustar · BYPASS → frase exata
