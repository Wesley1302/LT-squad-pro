# Scale Readiness Checklist (BLOQUEANTE)

- [ ] Winner validado: ROAS 2+ com 50+ conversões
- [ ] Estrutura inteira ≥ ROAS 2 (ou LTV comprovado justifica ROAS mais apertado, ex: 1.6-1.7)
- [ ] CAC aceitável (< ticket)
- [ ] Criativos winners existem (≥1, ideal vários)
- [ ] Página validada
- [ ] Volume suficiente de dados
- [ ] LTV compreendido
- [ ] Orçamento de escala = 10x ticket
- [ ] Regra dia 1-2 documentada (sem venda -> pausa; venda com ROAS baixo -> +24h olhando histórico; reduzir orçamento antes de pausar; média >= ~1,5 conforme esteira)

**Decisão:** PASS → WLT-build-scale-plan/próxima oferta · FAIL (not_ready) → WLT-funnel-diagnostician · BYPASS → "quero pular com risco documentado"
