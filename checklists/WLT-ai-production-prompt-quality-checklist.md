# AI Production Prompt Quality Checklist (BLOQUEANTE)

- [ ] Usa artefato APROVADO (validation_status=pass)
- [ ] Define objetivo
- [ ] Define plataforma alvo
- [ ] Inclui contexto suficiente (do state)
- [ ] Inclui inputs
- [ ] Inclui restrições
- [ ] Inclui output esperado
- [ ] Inclui critérios de aceite
- [ ] Inclui checklist final embutido
- [ ] NÃO inventa estratégia
- [ ] NÃO contradiz oferta / produto / avatar

**Decisão:** PASS → entrega o prompt · FAIL → recusar e devolver ao agent dono do artefato · BYPASS → não se aplica (bloqueio de qualidade)
